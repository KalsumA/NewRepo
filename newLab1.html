Course Catalog Description:

This course will cover the creation of Web pages and sites using HTML, CSS, Javascript and graphical applications. Networked multimedia distribution technologies are also explored. The designof effective Web design including page layout, user interface design, graphic design, cntent flow, and site structure as well as management of Wed site resouces including intranet management and design considerations are addressed. Students design and create a major Web site wih multiple pages and cross-linked structures. Prerequisites: [none] Credit: 2-2-3 Semester Hours

Course Outcomes:

Students completing this course will be able to:

  • Recognise HTML, CSS, and Javascript mark-up and code a web page/application
  • Select the proper mark-up tags or code t acieve a particular result
  • Produce moder standards compliant web pages.
  • deploy web pages to a public server

Course objectives:

At the conclusion of this course, each successsful student will be able to:

  • Explain the client and server architecture of the Internet and related web technologies.
  • Use a basic text editor and other software tools to create web pafes using HTML, CSS< Javascript, jQuery, and images
  • Deploy web pages to a Linux-based web server using sftp
